## The twelve anti-ugliness laws

1. **Silhouette first — passed.** Emberlight’s 32px flat-black harness is snapshot
   locked; every other world owns distinct authored silhouette contracts and comparative
   fingerprints.
2. **No unresolved geometry — passed.** The live sweep found no partial-load shapes;
   orbit scaffolds and random-star regressions are explicitly forbidden by tests.
3. **No naked primitive — passed.** Generic rotated stacking is frozen to three legacy
   chamber pieces; the seven flagship worlds use authored set-piece and presentation
   registries.
4. **The two-second test — passed.** Every visible object binds a phenomenon, purpose,
   source name, lore relationship, and semantic label through the manifest audit.
5. **One bloom — passed.** `WorldRenderer` owns one quality-tiered bloom canvas and one
   screen-space composite; no object owns a second post-process bloom pass.
6. **Intentional size rhythm — passed.** Dense-pack layout properties enforce stable
   equal grids or structural landmark scaling instead of incidental near-sizes.
7. **Nothing floats without an address — passed.** Every object has an authored screen
   zone, overlap group, Heart clearance, and universe-specific physical address.
8. **At least 35% void — passed.** The salience governor hides ambient demand when the
   luminous coverage budget would violate the void floor.
9. **Rotation means something — passed.** Motion belongs to the frozen mechanical,
   tidal, growth, optical, atmospheric, waveform, or authored grammar; the live sweep
   found no arbitrary tilted-card scatter.
10. **Opacity is not a style — passed.** Translucency remains confined to water, gas,
    ghosts, light, reduced states, and glass overlays; solid set pieces remain solid.
11. **Count changes structure — passed.** All 126 Kindlings own distinct 1/10/25/50/100
    structural states, and high density merges into landmarks instead of duplicating
    sprites.
12. **Text never fights the world — passed after one fix.** World prose uses vignettes
    or glass. Phase 8.4 moved desktop notifications below top universe instruments and
    added an explicit narrow-layout override so the desktop shop-clear rule cannot push
    mobile notifications off-screen.
