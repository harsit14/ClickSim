# Phase 3 world-object cards

These are existing Archive objects, not new economy objects. Their manifest remains the source of truth for phenomenon, purpose, zone, material, motion, sound, lore, reduced-motion, and low-quality fields. Phase 3 changes only their in-world form.

Shared interaction card: each object occupies its manifest `near` or `far` zone to make a collected record visible without entering the Heart target; it idles with the manifest motion, reveals its full observation/implication/effect on focus or hover, and opens the local Archive on activation. Archive objects are binary collected records, so the 1/10/25/50/100 count thresholds are not applicable. Reduced-motion and low-quality forms use the object’s validated manifest fallbacks. Hit targets are 4.1rem, larger than the retired 3.15rem badges.
